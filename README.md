# My Neovim setup

Based on the starter template for [LazyVim](https://github.com/LazyVim/LazyVim).

- [config](/lua/config/README.md) - sets core vim and LazyVim configurations
- [plugins](/lua/plugins/README.md) - adds plugins and augments default configs
- [`lazyvim.json`](./lazyvim.json) - captures extras that have been added (modified via `:LazyExtras`)
